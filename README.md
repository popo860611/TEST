# Artale Helper

Simple automation helper for MapleStory Artale written in Python.

This project is inspired by the approach used in
[MapleStoryAutoLevelUp](https://github.com/KenYu910645/MapleStoryAutoLevelUp).

## Structure

- `main.py` - GUI entry point
- `src/` - automation logic and configuration
- `ui/` - optional `.ui` files
- `assets/` - your sample images

## Requirements

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python main.py
```

Refer to the original project for more detailed setup tips and
game-specific notes.
